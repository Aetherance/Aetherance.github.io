TASK

打造一个绝无伦比的 xxx-super-shell (xxx 是你的名字)，它能实现下面这些功能：

  • 实现管道（也就是 |）
  • 实现输入输出重定向（也就是 < > >>）
  • 要求在管道组合命令的两端实现重定向运算符

PLAN

  • 实现后台运行（也就是 &）
  • 实现 cd，要求支持能切换到绝对路径、相对路径和支持 cd -
  • 屏蔽一些信号（如 ctrl + c 不能终止）
  • 界面美观
  • 开发过程记录、总结、发布在个人博客中
    要求：
  • 不得出现内存泄漏、内存越界等错误 - 学会如何使用 gdb 进行调试，使用 valgrind 等工具进行检测
